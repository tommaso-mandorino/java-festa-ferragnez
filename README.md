# Esercizio "Festa Ferragnez"

## Contesto
State lavorando con il servizio di sicurezza dei Ferragnez e il vostro compito è di assicurarvi che accedano alla festa solo gli invitati presenti sulla lista.

## Passaggi
1. Creare un progetto `java-festa-ferragnez` con il package `com.ferragnez.party`, a cui aggiungere la classe `CheckGuest`.
2. **Creare** e **inizializzare** l’**array** contenente i nomi degli invitati.
3. **Chiedere** all’utente come si chiama.
4. **Verificare** che il nome sia presente nella lista.
5. **Lasciarlo entrare** o **rispedirlo cortesemente da dove è venuto**.

## Lista invitati
Dua Lipa, Paris Hilton, Manuel Agnelli, J-Ax, Francesco Totti, Ilary Blasi, Bebe Vio, Luis, Pardis Zarei, Martina Maccherone, Rachel Zeilic.

## Bonus
In caso di utilizzo del **ciclo for** per cercare il nome nella lista nella consegna base, implementare lo stesso programma usando il **ciclo while**, e viceversa.
